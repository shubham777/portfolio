Hurricane_harvey_UAV-Image-Segmentation


Objective:

The project aimed to develop a semantic image segmentation model leveraging remotely sensed data post-Hurricane Harvey in Houston, Texas. The focus was on accurately identifying common assets in residential areas by segmenting high-resolution photographs captured by a small UAV. The dataset consisted of 299 images with dense segmentation masks containing 27 classes like Property Roof, Swimming Pool, Vehicle, etc.

Methodology:

  1. Data Augmentation: Enhanced dataset quality using image augmentations: horizontal flip, brightness/contrast/color adjustments, blurring, sharpening, Gaussian noise, and normalization.

  2. Semantic Segmentation Modelling: Trained on a dataset of 299 images with dense segmentation masks containing 27 classes like Property Roof, Swimming Pool, Vehicle, etc.For this I employed a UNet model with Fully Convolutional Network (FCN) architecture to achieve high-resolution image segmentation.

  3. Model Evaluation: Ran multiple models with varied parameters and assessed their performance using accuracy and loss curves. The SeResNet model emerged with a notable 62.65% accuracy on the test set predictions.

Impact and Significance:

The project represents a significant advancement in post-disaster response and recovery through state-of-the-art technology and advanced machine learning algorithms.It further holds potential to enhance response efficacy and timeliness in natural disasters and human-caused conflicts, paving the way for improved disaster response efforts.

Skill Acquisition:

During the project, I acquired skills including augmentation techniques and deep learning-based image segmentation, alongwith enhacing my skills for data preprocessing, model evaluation and selection.

Conclusion:

The project opens avenues for further advancements in image segmentation models and disaster response technologies, driving innovation in disaster management. Furthermore, it not only demonstrates the potential of sophisticated machine learning algorithms in disaster response but also highlights the acquired skills and knowledge vital for future developments in this crucial domain.
