Predicting AirBnB Prices in New York


Description:

I spearheaded a transformative project aimed at predicting Airbnb listing prices in New York City, harnessing regression models and a comprehensive dataset containing over 48,000 listings. This dataset, sourced from Kaggle, encompasses critical details such as location, room type, reviews, availability, and host information.

Key Project Phases:

  • Exploratory Data Analysis (EDA): Utilized powerful visualization techniques to understand feature distribution and relationships, guiding feature selection for modeling and visualized correlations among numerical features using Seaborn heatmap.

  • Data Preprocessing: Addressed outliers in the target variable and handled missing values in alongwith conducting feature engineering and feature scaling.

  • Modeling: Explored various regression models including Classic ML models (ridge regression, lasso regression, decision tree regression). Also, implemented ensemble methods (random forest, extra trees, gradient boosting, XGBoost, CATBoost).

  • Model Evaluation: Evaluated models using metrics such as r2 score, MSE, MAE, and RMSE on test data and utilized cross-validation to gauge model generalization performance.

  • Hyperparameter Tuning: Leveraged random search to optimize hyperparameters for each model, enhancing overall performance.

  • Visualizing Results: Utilized Yellowbrick library to visualize and compare model performance across various evaluation metrics.

Key Contributions and Impact:

I leveraged powerful visualization techniques and exploratory data analysis to identify significant correlations and patterns within the dataset, contributing to the development of a comprehensive understanding of the factors influencing Airbnb listing prices in an urban setting. Furthermore, I was successful in showcasing the superiority of ensemble methods, particularly highlighting the accuracy and robustness of models like XGBoost and CATBoost.

Future Implications:

This project’s success underscores the potential of data-driven approaches in optimizing pricing strategies within the hospitality sector. By extrapolating these findings to other urban areas or accommodation types, it has the potential to revolutionize pricing methodologies, benefiting both hosts and guests. Unlocking the secrets behind Airbnb pricing in New York City was the goal of this project.
